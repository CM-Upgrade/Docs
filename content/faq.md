Common questions about UpgradeMate licensing, version coverage, and demo limits. [Contact us](https://www.upgrademate.io/contact) if your question isn't covered here.

<div class="faq-list">

{{% details title="I purchased a paid license, but it only covers one Windows version. Do I need to buy a separate license for other Windows versions?" closed="true" %}}
No. During your one-year license period, you can request additional licenses for other Windows versions on the same SCCM site (Support ID) at no extra cost. Reply to your original activation email and tell us which Windows version you need a license for.
{{% /details %}}

{{% details title="Can I use UpgradeMate to upgrade to more than one Windows version at the same time?" closed="true" %}}
A single UpgradeMate installation supports one Windows upgrade target at a time. Once that rollout is complete, [remove the existing configuration]({{< ref "operations/license-management-and-removal.md" >}}) and [run the setup wizard again]({{< ref "getting-started.md" >}}) to start the next upgrade.
{{% /details %}}

{{% details title="My demo license expired. Can I request another one?" closed="true" %}}
Demo licenses are limited to one per SCCM site (Support ID). To continue using UpgradeMate after a demo, [contact us](https://www.upgrademate.io/contact) to purchase a paid license.
{{% /details %}}

</div>
