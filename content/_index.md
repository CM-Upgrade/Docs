## What is UpgradeMate?

{{< param "productName" >}} is the smart add-on for ConfigMgr (SCCM) that turns
enterprise Windows OS upgrades from a nightmare into a smooth ride. A guided
wizard provisions everything the rollout needs — collections, task sequences, OS
upgrade packages, an end-user form and a Power BI dashboard — so you can move
from a small pilot to a fleet-wide upgrade without scripting any of the plumbing
yourself.
